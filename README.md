# CallTrainer

An AI-powered training application that helps users improve their telephone communication skills through simulated conversations, speech and communication analysis, and qualitative feedback on speaking behavior over time.

Developed as part of a university project seminar (PJS) in collaboration with **SOLOX GmbH**.

## Motivation

Most communication training focuses on domain expertise or single numeric scores. CallTrainer instead focuses on:

- **Communication skills over domain knowledge** — training how something is said, not just what
- **Qualitative feedback over single scores** — actionable insights instead of a bare number
- **Recognizing blind spots over time** — surfacing recurring patterns in speaking behavior across sessions
- **Simplicity by default** — a clean core UI with advanced settings tucked away for users who want them

## Key Features

| Category | Examples |
|---|---|
| Conversation Simulation | Realistic simulated phone conversations with adjustable scenarios |
| Speech & Communication Analysis | Analysis of speaking patterns, tone, and communication style |
| Feedback & Evaluation | Qualitative, behavior-focused feedback after each session |
| Learning Process | Progress tracking across sessions to reveal long-term patterns |
